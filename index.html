English Learning App

A Full Stack Web Application for Non-English Speakers

Graduation Project Presentation

Project Overview

  • Target Audience: Non-English speakers
  • Key Features: Interactive flashcards, exercises, and automatic feedback
  • Learning Approach: Word recognition, context usage, and practical application
  • User Interaction: Fill-in-the-blank exercises, immediate feedback
  • Multilingual Support: Interface available in multiple languages

Technology Stack

  • Frontend: Vue.js, Vuex (State Management), Vue Router
  • Backend: Node.js, Express.js
  • Database: MongoDB (User data storage)
  • Authentication: JSON Web Tokens (JWT)
  • Internationalization: vue-i18n
  • Build Tool: Vite

Project Architecture

Frontend (Vue.js) ↔ Backend (Node.js/Express) ↔ Database (MongoDB)

API Endpoints

  • POST /api/auth/register - User registration
  • POST /api/auth/login - User login
  • GET /api/lessons - Retrieve all lessons
  • GET /api/lessons/:id - Get specific lesson
  • POST /api/lessons/:id/submit - Submit lesson answers
  • GET /api/user/profile - Get user profile
  • PUT /api/user/profile - Update user profile

Data Flow

User Experience

  • Intuitive interface for easy navigation
  • Progressive learning path
  • Real-time feedback on exercises
  • Personalized learning recommendations
  • Multi-device support for learning on-the-go

Security Measures

  • Secure user authentication with JWT
  • Data encryption for sensitive information
  • Regular security audits and updates
  • HTTPS protocol for all communications
  • Rate limiting to prevent abuse

Conclusion

We hope to expand this project to include more than 100 lessons and fully develop it into a real application for non-English speakers around the world.

With continued updates and improvements, our goal is to provide a valuable resource for users globally, making English learning more accessible and effective.
