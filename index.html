Project Overview

Brief Description: An English learning platform with user management, lesson tracking, and a chatbot.

Purpose: To provide an interactive and engaging way for users to learn English.

Architecture and Technologies

Backend: Django, Django REST Framework

Database: PostgreSQL

Authentication: JWT (JSON Web Tokens)

Third-Party Services: Google OAuth2 for authentication, SMTP for email
